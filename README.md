# 🧹 Kamer Bedi

### কাজের মানুষ, সম্মানের মানুষ

একটি সম্পূর্ণ বাংলা ওয়েবসাইট যা দক্ষ কর্মী এবং কর্মপ্রার্থীদের মধ্যে সেতুবন্ধন তৈরি করে।

---

## 📖 প্রকল্পের বিবরণ

**Kamer Bedi** একটি প্ল্যাটফর্ম যেখানে:
- গৃহস্থরা তাদের প্রয়োজন অনুযায়ী দক্ষ কর্মী খুঁজতে পারেন
- কর্মীরা তাদের দক্ষতা অনুযায়ী কাজের সুযোগ পেতে পারেন
- ঘর সামলানো, রান্না, শিশু যত্ন, বয়স্ক সেবা, ড্রাইভিং, কারিগরি কাজ সহ বিভিন্ন সেবা প্রদান করা হয়

---

## 🛠️ প্রযুক্তি

- **HTML5** - স্ট্রাকচার
- **CSS3** - ডিজাইন ও স্টাইলিং (কাস্টম CSS, ফ্লেক্সবক্স, গ্রিড)
- **JavaScript** - ইন্টারঅ্যাক্টিভিটি (ফর্ম হ্যান্ডলিং, অ্যানিমেশন, ফিল্টারিং)
- **Vercel** - হোস্টিং ও ডেপ্লয়মেন্ট
